# cli-github-api-go

CLI de uso para interactuar con la api de Github.

## Instalación

1. Clona el repositorio:

   ```bash
   git clone <URL-del-repositorio>
   ```

2. Navega al directorio del proyecto:
   ```bashs
   cd <nombre-del-directorio>
   ```

## Ejecución

Para ejecutar el proyecto localmente:

```bash
go run main.go
```
